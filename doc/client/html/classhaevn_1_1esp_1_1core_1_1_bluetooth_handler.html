void BluetoothHandler::deviceDiscovered ( const QBluetoothDeviceInfo &  deviceInfo)
slot

deviceDiscovered displays a device when it gets discovered

This slot will be invoked when discoveryAgent discovers a bluetooth device nearby. The device will then be displayed on the dialog. It also checks if a device is already displayed, so a device only exists one time in the list. If a device is paired or authorized paired it gets displayed in the color green and if a device is not paired it gets displayed blue.

void BluetoothHandler::showPairingMenu ( const QPoint &  pos)
slot

showPairingMenu shows menu when clicked on a device in a list

This slot will be invoked when clicked on a device in the bluetooth list. It show two Actions: "Pair..." and "Unpair...". If clicked on "Pair..." it connects to the bluetooth device. And if clicked on "Unpair..." it removes the connection to the bluetooth device.
